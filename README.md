# 🤖 AI ChatBot

A smart, interactive AI-powered chatbot built using **HTML**, **CSS**, **JavaScript**, and **Python Flask**. This chatbot responds to user messages with AI-generated replies based on pre-trained text data using NLP (Natural Language Processing) techniques such as **TF-IDF** and **Cosine Similarity**.

## 🌟 Features

- 💬 Clean and modern chatbot interface
- 🔁 Real-time messaging with auto scroll
- 🚀 Press "Enter" to send messages instantly
- 🤖 NLP-powered response generation using custom AI logic
- 🧠 Learns from `chatbot.txt` — easily expandable
- 🔐 CORS support enabled for frontend-backend communication

---

## 🧱 Tech Stack

### Frontend
- HTML
- CSS (Glassmorphism style)
- JavaScript (Fetch API)

### Backend
- Python 3
- Flask
- Flask-CORS
