# 🤖 AI ChatBot

An interactive AI-powered chatbot built using **HTML**, **CSS**, **JavaScript**, and **Python Flask**. It uses NLP techniques to understand user queries and respond intelligently using a trained dataset (`chatbot.txt`).

---

## 📌 About This Project

AI ChatBot is an interactive web-based chatbot application that uses Natural Language Processing (NLP) techniques to understand and respond to user queries. Built using Python (Flask) for the backend and HTML/CSS/JavaScript for the frontend, this chatbot simulates a conversational AI by analyzing the input and returning the most relevant response from a predefined dataset.

The core logic is powered by TF-IDF (Term Frequency–Inverse Document Frequency) and Cosine Similarity, which help the bot identify the most appropriate answer from the chatbot.txt training file.

This project is beginner-friendly and a great example for learning how to:

* Build a chatbot using basic AI techniques

* Integrate Python backend with a web-based frontend

* Deploy a simple Flask app for real-time interaction
